## Общая память (граф знаний)

### Концепции
- **1С-сообщество** [concept]: Целевая аудитория разработчиков
- **Agent SDK** [tool]: Фреймворк для создания интеллектуального агента
- **Agent SDK кредит** [concept]: 100 долларов действителен до 15.06.2026
- **Anthropic CCR** [tool]: Облачные расписания для автоматизации (5 задач)
- **B2B аналитика** [decision]: Услуга для вендоров софта
- **CLAUDE.md** [tool]: Правила совместной разработки
- **GitHub репозиторий** [project]: github.com/darachubarova/infostart-agent
- **Kuzu** [tool]: База данных для графа общей памяти
- **SEO-платформа** [project]: Платформа поиска и оптимизации для 1С-разработчиков
- **White-label контент** [plan]: Переиспользуемый контент для партнёров
- **Windows Task Scheduler** [tool]: Локальное расписание (5 задач)
- **article.docx** [tool]: Word-документ через pandoc конвертер
- **article.md** [tool]: Авторская статья в markdown
- **pandoc** [tool]: Конвертер markdown в docx
- **trend_score** [concept]: Метрика count × avg_comments для ранжирования
- **Агент анализа трендов** [project]: Agent SDK для мониторинга infostart.ru
- **Граф памяти** [concept]: Структура знаний и связей контента
- **Исключение выбросов** [plan]: Фильтрация вирусных контентов
- **Контент-машина** [project]: Система автоматической генерации контента для 1С-сообщества
- **План MAX x5** [concept]: Тариф с кредитом 100 USD
- **Платная подписка** [decision]: Доступ для HR-рекрутеров
- **Подписка MAX x20** [decision]: Монетизационная цель (увеличение доходов в 20 раз)
- **Режим Forum** [plan]: Создание статей-ответов на неразрешённые вопросы форума
- **Режим Trends** [plan]: Генерация трендовых статей из топа infostart.ru
- **Сбор статей** [plan]: Извлечение 5 страниц с платформы
- **Статья SOS** [decision]: Публикация о государственных интеграциях с 436 комментариями
