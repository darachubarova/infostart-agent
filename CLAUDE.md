# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Общая память

В начале каждой сессии читай `memory/context.md` — это граф знаний из всех предыдущих бесед. Он обновляется автоматически после каждой сессии через `python memory/extract.py`.

Чтобы добавить текущую сессию в граф вручную:
```powershell
python memory/extract.py "краткое резюме сессии"
```

## Запуск

```powershell
npm start   # node --env-file=.env --import tsx agent.ts
```

`ANTHROPIC_API_KEY` хранится в `.env`. Node 24+, ES modules (`"type": "module"`).

## Архитектура

`agent.ts` — единственная точка входа. Запускает вложенный агент через `@anthropic-ai/claude-agent-sdk`, который:

1. Загружает 5 страниц `infostart.ru/1c/articles/?period=30&sort=property_comments` через `WebFetch`
2. Сохраняет сырые данные в `raw_data.json`
3. Вычисляет `trend_score = count × avg_comments` по категориям, исключая выбросы (>3× среднего)
4. Сохраняет анализ в `trends.json`
5. Пишет авторскую статью в `article.md`

Вложенный агент работает с инструментами `WebFetch`, `Write`, `Read`, `maxTurns: 25`.

## Стиль генерируемых текстов

Приоритет: **точность → живость авторского голоса → скорость**.

Статьи — от первого лица, коллега разговаривает с коллегой. Без маркированных списков, канцелярита, шаблонных фраз («таким образом», «следует отметить»). Конкретные примеры и заголовки статей как доказательства тезисов.

## Стиль совместной работы

- Язык общения: **русский**
- Автономия: **полная** — действуй самостоятельно, не спрашивай подтверждения на каждый шаг
- Будь проактивен и целеустремлён: если видишь лучший путь к цели — иди им
- Пользователь доверяет твоей экспертизе — предлагай решения, а не варианты
